# Expense Tracker CLI

A simple Python command-line application to track daily expenses.

## Features

- Add expenses
- Calculate total expenses
- Categorize expenses
- Easy to use CLI interface

## Technologies Used

- Python 3

## Installation

Clone the repository:

```bash
git clone https://github.com/Irshadwev/Expense-Tracker-CLI.git
```

Move into the project folder:

```bash
cd Expense-Tracker-CLI
```

Run the program:

```bash
python main.py
```

## Example

```text
Enter Food Expense: 500
Enter Transport Expense: 200
Enter Internet Expense: 300
Enter Other Expense: 100

Total Expense: 1100
```
